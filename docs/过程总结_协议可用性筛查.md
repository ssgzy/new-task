## 当前应继续单列的模型
- `Lion-7B`：历史锚点 / 协议失败案例
- `MiniLLM-Llama-7B`：`base_distilled baseline`
- `LaMini-LLaMA-7B`：`pending_repo_id`

## 当前扩展候选的优先级结论
- `DeepSeek-R1-Distill-Qwen-7B` 和 `OpenR1-Distill-7B` 都还值得继续最小调试，但下一步都应先“查官方 wrapper”，不要继续盲试 parser 或随机 prompt。
- `Xwin-LM-7B` 当前不是“差一个 Answer 行”的问题，而是 `<unk>` 重复退化；如果继续投入，应先做 tokenizer/special-token 专项核查，而不是重复跑同一条 Vicuna smoke。

## 当前明确不恢复 canonical 主线
- 不恢复 canonical `val_calib50`
- 不恢复 canonical `val_screen200`
- 不改 parser
- 不把 benchmark 主协议改成 CoT
