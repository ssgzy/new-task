- 本轮先对 `main` 组执行准入筛选。
- 脚本现支持：
  - `--groups main`
  - `--labels "<模型名>"`
  - `--resume`
  - `--provisional`
  - `--screen-only`

## provisional 规则
- 单模型或临时检查结果默认写到 `outputs/provisional/`
- canonical `outputs/qualification_summary.csv` 只保留正式主表聚合结果

## 当前执行约束
- 在当前这一步，只执行主表 `val_screen200`
- 为避免提前进入 `validation883`，应使用：
  - `conda run -n Lion python scripts/run_qualification_v1.py --labels "<模型名>" --model-registry "<6模型registry>" --run-root outputs/provisional/screen200_6model_20260407 --report-csv outputs/provisional/screen200_6model_20260407/reports/<模型slug>.csv --max-new-tokens 256 --resume --screen-only --provisional`
- 每跑完一个模型，立刻刷新：
  - `scripts/build_screen200_table_v1.py`
