# 常用命令

## 环境
```bash
conda run -n Lion python --version
conda run -n Lion pip list
conda run -n Lion python -c "import torch, transformers, huggingface_hub, sentencepiece, tiktoken, google.protobuf; print(torch.__version__); print(transformers.__version__); print(huggingface_hub.__version__); print(sentencepiece.__version__); print(tiktoken.__version__); print(google.protobuf.__version__)"
```

## 项目根目录
```bash
cd "/Users/sam/Documents/Documents - sam的MacBook Pro/论文📑/Distillation/new task"
```

## 一次性环境安装
```bash
conda create -n Lion python=3.11.14 -y
conda run -n Lion pip install -r requirements-benchmark.txt
```

## FinQA 重拉与字段核对
```bash
conda run -n Lion python scripts/prepare_finqa_data.py
conda run -n Lion python scripts/prepare_finqa_data.py --overwrite
```

## 查看元数据
```bash
sed -n '1,120p' outputs/metadata/finqa_split_summary.csv
sed -n '1,220p' outputs/metadata/finqa_validation_field_summary.json
sed -n '1,120p' outputs/metadata/finqa_manifest_index.csv
```

## 导出 protocol_v1
```bash
conda run -n Lion python scripts/finqa_protocol_v1.py
```

## 模型注册与下载
```bash
conda run -n Lion python scripts/ensure_candidate_models.py
conda run -n Lion python scripts/ensure_candidate_models.py --download-missing
conda run -n Lion python scripts/ensure_candidate_models.py --labels "Orca-2-7B" --download-missing --provisional
```

## 主表正式运行
```bash
conda run -n Lion python scripts/run_length_calibration_v1.py --groups main --resume
conda run -n Lion python scripts/run_qualification_v1.py --groups main --resume --screen-only
```

## 单模型 provisional 运行
```bash
bash scripts/run_assigned_model_v1.sh --label "Orca-2-7B" --download-model
conda run -n Lion python scripts/run_length_calibration_v1.py --labels "Orca-2-7B" --resume --provisional
conda run -n Lion python scripts/run_qualification_v1.py --labels "Orca-2-7B" --resume --provisional --screen-only
```
