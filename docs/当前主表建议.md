## 2026-04-07 最新执行决策
- 当前 `validation883` 执行名单从 `4` 个正式通过模型，扩展为 `5` 个报告模型：
  - `Orca-2-7B`
  - `Mistral-7B-Instruct-v0.3`
  - `Qwen2.5-7B-Instruct`
  - `Yi-1.5-6B-Chat`
  - `ChatGLM3-6B`
- 当前口径：
  - 前 `4` 个仍是正式通过 `screen200` 的 main-table 模型
  - `ChatGLM3-6B` 作为 `boundary-pass candidate` 一并报告 `validation883`
- 纳入理由：
  - `valid_parse = 0.595` 与门槛 `0.60` 只差 `0.005`
  - 换算到 `200` 条 screening，只差约 `1` 条样本
  - `ChatGLM3-6B` 是当前候选中唯一的 `GLM / Prefix LM` 路线，也是唯一需要 `trust_remote_code` 的非标准 decoder-only 路线
  - 如果只保留 `4` 个标准 decoder-only Transformer，方法覆盖面会变窄

- `Zephyr-7B-beta`
  - 当前定位：`negative case`
  - 结论：
    - 不进正式主表
    - 适合作为 `DPO-only` 在当前 `direct-answer` 协议下的负例分析
